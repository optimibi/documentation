# OLAP设置


> **联机分析处理**(On Line Analytical Proccessing，简称OLAP) 概念最早由关系数据库之父E.F.Codd于1993年提出。OLAP应用是目前数据仓库上的重要应用之一，是决策分析的关键。

Datafor内置了一个OLAP引擎，根据项目的不同场景，可以配置OLAP。
